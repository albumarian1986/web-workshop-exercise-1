Programming C# in VS Code

VS Code is a lightweight, powerful editor for C# when paired with the C# (OmniSharp) extension and the .NET SDK.

Quick Start

  1. Install the .NET SDK from dotnet.microsoft.com.
  2. Install the C# (OmniSharp) extension in VS Code.
  3. Open the folder in VS Code and use the integrated terminal to run dotnet run.

Hello World (example)

Save the example below as HelloWorld.cs inside a console project or use the provided example in the examples/ folder.

using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hello, VS Code + C#!");
    }
}

# FirstDemo

A tiny demo website showing a short intro and tips for programming in Visual Studio Code.

## Files

- `index.html` — main page.
- `css/styles.css` — page styling.
- `js/script.js` — page interactivity.

## Preview

Open `index.html` in a browser. For live reload use the VS Code Live Server extension and click "Go Live".

## Notes

- The page now uses separate files for HTML, CSS, and JavaScript.
- The gallery images are stored as SVG files in the `assets/` folder.
- To edit, open the project folder in VS Code and update `index.html`, `css/styles.css`, `js/script.js`, or the asset files in `assets/`.

## C# Example

- A simple C# example is included in `examples/HelloWorld.cs`.
- To run the example:

	1. Install the .NET SDK: https://dotnet.microsoft.com/
	2. Create a new console project or place `HelloWorld.cs` inside a project and run:

```powershell
dotnet run
```

3. Install the C# extension (OmniSharp) in VS Code for language features, debugging, and IntelliSense.
